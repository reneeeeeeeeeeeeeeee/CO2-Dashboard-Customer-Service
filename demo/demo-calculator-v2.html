Customer Service CO2 Calculator

Voice Bot Formula
per_voice_session_kWh = voice_GB_per_min x network x duration_min + llm_turns x llm_kWh_per_turn + customer_power x duration_min / 60
handover_add_on = transferred_sessions x call_CO2e — each transfer adds the cost of the resulting call; voice session energy is already counted in sessions x per_session_CO2e
resolution_overhead = resolved_sessions x voice_session_CO2e x 0.08 — infrastructure overhead on successful paths, not a net saving
intent_failure_rate = (sessions - resolved_sessions) / sessions — share of sessions that did not resolve autonomously

Method Notes

          
How V2 Thinks

          

            

              1. Transparent defaults
              
The calculator starts from documented defaults already present in the repo and exposes them as editable assumptions.

            

            

              2. Aggregated first
              
This demo computes from monthly channel totals. It is intentionally lighter than an event-level engine.

            

            

              3. Net effect, not slogan
              
Automation is measured as operating cost minus avoided human load, not assumed to be green by default.

            

          

          

            Recommended wording stays the same as the rest of the repository: estimated operational CO2 impact, activity-based estimate, documented assumptions, and stated reporting boundary.
